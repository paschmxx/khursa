

Audio/Video App

An audio / video app built using queries and database assets A user should be able to log in, filter and select a movie, tv show or audio based on user profile (adult or kids). An API will be built that will let a user comment on the selection, rate it, and share via social media.

Getting Started

Prerequisites You’ll need Git. See detailed install instructions here., npm, node, MAMP/WAMP.

Setup Navigate to htdocs in MAMP.

git clone https://github.com/paschmxx/Pashchenko_M_AV_App.git

Open the folder in your command line, and install the needed dependencies:

cd Pashchenko_M_AV_App npm install

Finally, run npm start to run the Sass compiler. Then navigate your browser to http://localhost:8888/Maksym_Pashchenko_AV_App The site will not show properly unless these steps are followed.
